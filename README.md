pscara-analyse
pscara-analyse is a DUAL LICENSED power systems compliance library that allows industry wide collaboration at a low cost. 
Companies only need support 1 psCARA license to gain access to pscara-analyse, thus it is a very low cost medium for collaboration on power systems compliance analysis.
